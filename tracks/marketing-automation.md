# 营销自动化与线索获取 Skill 路线

适合：增长负责人、营销 Ops、B2B 销售运营、内容分发团队。

## 目标

把表单、CRM、邮件、社媒、数据报表和内部提醒串起来，减少手工搬运；但保留关键审批和人工抽检。

## 推荐组合

| 场景 | 推荐 Skill / 应用 | 用法 |
|---|---|---|
| 工作流编排 | [`n8n`](../skills/n8n.md) / [`Activepieces`](../skills/activepieces.md) | 表单、CRM、邮件、飞书/Slack、报表联动 |
| 线索研究 | Apollo / LinkedIn / Google Search | 找目标公司、联系人、行业关键词 |
| 邮件与 Newsletter | Gmail / Mailchimp / ConvertKit | 草稿、分组、发送、效果复盘 |
| 社媒分发 | LinkedIn / X / Reddit / YouTube | 发布草稿、评论监控、内容表现跟踪 |
| 数据分析 | [`Plausible`](../skills/plausible-analytics.md) / GA / GSC | 页面、来源、关键词和转化趋势 |

## 安全边界

- 客户名单、外联内容、报价、承诺、合同信息必须人工确认。
- API Key、Cookie、私域数据不能写入公开仓库。
- 不做骚扰式外联、刷量、绕过平台限制。
