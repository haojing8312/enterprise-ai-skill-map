# 内容生产与分发系统 Skill 路线

适合：个人 IP、内容团队、出海内容运营、Newsletter 运营者。

## 推荐闭环

```text
公开信号监控 → 选题池 → 研究与草稿 → 编辑复核 → 发布/分发 → 数据复盘
```

## 推荐 Skill

- [`RSSHub`](../skills/rsshub.md)：公开信号订阅、竞品和行业动态监控
- [`Ghost`](../skills/ghost.md)：博客、Newsletter、内容中心
- [`n8n`](../skills/n8n.md)：把选题、表格、通知、分发串起来
- [`Activepieces`](../skills/activepieces.md)：可控自动化连接层
- [`Plausible`](../skills/plausible-analytics.md)：内容访问和转化复盘

## 注意

内容类 Skill 最怕“模板稿”和“自动发布”。建议默认保留人工编辑环节：标题、事实、引用、CTA、客户信息和价格承诺都要复核。
