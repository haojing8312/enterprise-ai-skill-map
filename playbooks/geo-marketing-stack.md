# GEO Marketing Stack｜出海 GEO / SEO 内容增长组合方案

## 最小闭环

```text
公开信号源 → 选题池 → 内容阵地 → 分发提醒 → 数据复盘
```

## 推荐组件

1. [`RSSHub`](../skills/rsshub.md)：监控公开来源。
2. [`n8n`](../skills/n8n.md) / [`Activepieces`](../skills/activepieces.md)：把信号写入表格、知识库或飞书提醒。
3. [`Ghost`](../skills/ghost.md)：博客、Newsletter、内容中心。
4. [`Plausible`](../skills/plausible-analytics.md)：看页面访问和转化趋势。
5. [`app-automations`](../app-automations/README.md)：连接 GSC、Apollo、LinkedIn、X、Mailchimp、Gmail 等应用。

## 推荐字段

选题池至少包括：

```text
来源｜标题｜链接｜目标市场｜目标关键词｜适合内容角度｜证据等级｜风险备注｜下一步动作
```

## 验收标准

- 每天能自动进入 10–30 条候选信号。
- 每周能筛出 3–5 个值得写的内容主题。
- 所有对外发布内容都有人工编辑复核。
- 数据复盘能看到来源、页面、关键词或转化趋势。
