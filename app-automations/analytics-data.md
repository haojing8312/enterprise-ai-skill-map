# Analytics & Data App Automations

适合：内容增长复盘、落地页分析、GEO/SEO 效果观察、营销报表。

## 常见应用

| 应用 | 用法 |
|---|---|
| Plausible | 隐私友好的页面访问和来源分析 |
| Google Analytics | 站点流量、事件和转化分析 |
| Google Search Console | 搜索词、页面表现、索引状态 |
| PostHog | 产品事件、漏斗、用户行为 |
| BigQuery | 更复杂的数据仓库分析 |
| Google Sheets / Airtable | 轻量数据表、选题池、日报归档 |

## 边界

数据只辅助判断。归因口径、事件定义、异常解释和业务结论必须人工确认。
