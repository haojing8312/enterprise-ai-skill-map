# Content & Social App Automations

适合：内容分发、社媒运营、Newsletter、个人 IP 和出海内容团队。

## 常见应用

| 应用 | 用法 |
|---|---|
| X / Twitter | 草稿优化、发布时间建议、互动信号观察 |
| LinkedIn | B2B 内容分发、公司页/个人页内容管理 |
| Reddit | 社区话题观察、公开讨论信号收集 |
| YouTube | 频道/视频/评论/订阅信号监控 |
| Instagram / TikTok | 内容分发表现和素材灵感 |
| Ghost / Webflow | 内容阵地、博客、Newsletter、落地页 |

## 边界

内容和社媒自动化只适合辅助：监控、草稿、提醒、复盘。正式发布、评论回复、私信触达默认人工确认。
