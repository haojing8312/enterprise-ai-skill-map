# 岗位｜开发工程师

读代码、改 bug、写测试、做小范围实现。

## 适用行业

- [软件/互联网/研发组织](../industries/software-internet.md)：4 个 Skill
- [AI/数据基础设施团队](../industries/ai-infrastructure.md)：1 个 Skill
- [内容/媒体/营销服务](../industries/content-media.md)：1 个 Skill
- [咨询/培训/知识服务](../industries/consulting-training.md)：1 个 Skill
- [零售/电商/客服密集型](../industries/retail-ecommerce.md)：1 个 Skill

## 岗位使用建议

- 把 Skill 当成工作流组件，不要当成完整岗位替代。
- 先写清输入、输出、审批人和失败回退方式。
- 能触达代码、账号、客户数据、合同或财务数据的能力，默认提高一个风险等级管理。
