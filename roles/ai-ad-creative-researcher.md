# AI 广告素材研究员 / AI Ad Creative Researcher

> 面向：跨境老板、市场负责人、投手、内容/设计负责人  
> 使用边界：公开广告库、公开创意中心、公开落地页、企业自有公开资料；不使用客户隐私、合同、报价、未公开产品或非授权广告账户数据。

## 这个岗位解决什么问题

广告预算每天都在花，素材会却经常卡在“竞品这样拍、设计再出几版、投手先测测”。AI 广告素材研究员要解决的不是自动投放，而是把素材会前的研究工作做扎实：哪些公开素材值得观察，哪些结构可以借鉴，哪些表达不能碰，下周应该给设计和剪辑什么 Brief。

## 岗位职责

AI 广告素材研究员负责从 Meta Ad Library、TikTok Creative Center、Google Ads Transparency Center、竞品公开落地页和公开视频里，持续拆解广告素材的 Hook、场景、卖点、CTA、落地页承接和风险项，再输出素材观察表、竞品卖点地图、原创素材 Brief 和测试假设。

## 适合谁

- 每月稳定投 Meta、TikTok、Google、YouTube 或独立站广告；
- 有明确竞品或对标品牌，公开广告可观察；
- 素材产能跟不上，设计/剪辑缺少清晰 Brief；
- 老板希望素材会有证据、有判断、有下一步动作。

## 可接手任务

1. 竞品广告库周采样；
2. Hook / 首屏文案 / 前 3 秒结构拆解；
3. 卖点与人群、场景、渠道映射；
4. 素材空白机会扫描；
5. 广告承诺与落地页承接检查；
6. 原创素材 Brief 批量生成；
7. 老板版素材趋势周报；
8. 素材资产库与复盘标签维护。

## 能力模块

| 模块 | 输出 | 人类验收 |
|---|---|---|
| 公开采样 | 公开广告链接、截图编号、国家、渠道、广告主、落地页 | 来源能打开，不绕过平台限制 |
| 结构拆解 | Hook、痛点、场景、产品出现时机、证据、CTA | 只基于输入材料，不推测效果 |
| 卖点地图 | 竞品 × 卖点 × 场景 × 渠道 × 国家矩阵 | 低样本标注，不夸大结论 |
| 原创 Brief | 脚本骨架、镜头清单、字幕方向、CTA、尺寸 | 不复制竞品素材，有不可复制元素说明 |
| 承接检查 | 广告承诺与落地页首屏、价格、FAQ、认证、售后是否一致 | 产品/合规负责人确认事实 |

## 工作机理

```text
确定 SKU / 国家 / 竞品 / 禁用声明
→ 公开广告库采样
→ 素材观察表
→ 拆解 Hook / 场景 / 卖点 / CTA
→ 检查落地页承接
→ 生成原创 Brief 与测试假设
→ 人工合规和产品事实审核
→ 设计剪辑制作
→ 投手小预算测试
→ 复盘标签回写
```

## 第一周试点

- **Day 1**：选 1 个主推 SKU、2 个目标国家、3–5 个竞品，写清禁用声明和可公开证明卖点；
- **Day 2**：采样 50–100 条公开广告和落地页，建立素材观察表；
- **Day 3**：深拆 15–25 条素材，做 Hook、场景、CTA 和卖点矩阵；
- **Day 4**：生成 8–12 个原创 Brief 和 5–8 个测试假设；
- **Day 5**：开 30 分钟素材会，选 3–5 条进入制作/测试，明确人工审核人。

## 可复制提示词

### 公开广告素材拆解

```text
你是 AI 广告素材研究员。请只基于我提供的公开广告链接、截图文字和落地页摘要进行分析，不要补充未给出的数据，不要推测广告预算或效果。

任务：把以下素材拆成结构化表格：广告主/品牌、渠道与国家、素材类型、Hook、目标人群或使用场景、核心卖点、证据或信任元素、CTA/优惠、落地页承接点、可借鉴结构、不可复制元素与风险、需要人工确认的产品事实。

输入素材：[粘贴公开链接、截图 OCR、落地页摘要]
```

### 生成原创素材 Brief

```text
你是跨境电商素材 Brief 助手。请把以下竞品素材“结构”改写成原创 Brief，禁止复制竞品文案、画面、商标、达人、音乐或专有素材。

我方产品事实：[粘贴已公开且可证明的卖点/参数/认证/禁用声明]
参考素材结构：[粘贴 Hook、场景、CTA、落地页承接，不粘贴原视频文件]

输出：Brief 标题、适用渠道与比例、目标人群、15 秒脚本、镜头清单、字幕方向、CTA、需要拍摄的素材、禁止出现的声明、需要人工确认的事实。
```

### 广告—落地页承接检查

```text
请检查以下广告承诺与落地页摘要是否一致。只基于输入内容，不访问后台，不推测转化。

输出表格字段：广告承诺 / 落地页是否承接 / 证据位置 / 风险等级 / 修改建议 / 需人工确认人。

广告素材摘要：[粘贴广告文案、CTA、优惠、截图 OCR]
落地页摘要：[粘贴首屏、价格、FAQ、认证、物流、售后、评论摘要]
```

## 相关 Skill / 工具栈

| 工具 / repo / 工作流 | 用途 | 风险边界 |
|---|---|---|
| Meta Ad Library / Ads Archive API | 查询公开 Meta 广告线索 | 优先官方 API 或人工检索，不绕过限制 |
| TikTok Creative Center | 观察 TikTok 高表现素材结构 | 不搬运素材，不把互动量当转化 |
| Google Ads Transparency Center | 查询 Google 生态公开广告 | 不推断预算、出价、ROI |
| Firecrawl / Crawlee / Playwright | 公开落地页快照、截图、结构化摘要 | 遵守 robots 和服务条款 |
| browser-use | 半自动网页研究流程 | 不输入敏感账号，不绕过验证码 |
| OpenAI Cookbook / CLIP | 结构化输出、标签、图文聚类 | 不接入隐私；相似度不是版权判断 |

## 人类主管验收

老板抽查来源链接；市场负责人看样本是否足够；产品负责人确认参数、认证、材质、保修、交期；投手确认测试假设；设计/剪辑确认 Brief 能开工；法务或合规负责人审核敏感声明。

## 不能交给 AI 自动决定

客户承诺、价格/报价、合同、预算、出价、账户策略、合规最终判断、版权/IP 判断、产品事实确认、平台限制绕过、效果承诺，都必须由人审。
