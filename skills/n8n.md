# n8n

一句话：适合把表单、CRM、邮件、报表这些增长运营环节接起来，减少手工搬运；对外发送、分群规则和触发条件必须人工抽检。

## 适合谁

- 营销增长负责人
- 内容运营负责人
- 出海营销负责人
- 自动化运营负责人

## 解决什么问题

给营销 Ops、增长负责人和内容分发团队用，适合做表单到 CRM、邮件、提醒、报表的流程编排；不要包装成自动骚扰或自动获客机器。

## 推荐场景

- 营销 Ops 流程编排
- 表单 → CRM → 邮件 → 报表链路
- 内容分发与线索同步

## 不适合

- 骚扰式外联
- 跳过审批的自动群发
- 对外二次分发前未做许可复核

## 风险边界

风险等级：`medium`

- fair-code / 非纯 OSI 许可，外部二次分发前需复核
- 对外触达流程必须有人审规则和频率
