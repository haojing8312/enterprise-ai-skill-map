# 研发/产品场景

## 场景问题

AI 写码真正的风险，不是代码生成慢，而是没有 spec、没有测试、没有验收，最后把错误交给人肉兜底。

## 可用 Skill

### obra/superpowers

- 类型：AI 编码工作流 / Skill 集合
- 适合：研发团队、技术负责人、AI coding assistant
- 解决：把计划、测试、排障、验收前置
- 边界：它是工作习惯和流程样本，不是“接上就自治”的神器
- 链接：https://github.com/obra/superpowers

## 可组合数字员工

- product-manager：写 spec 和验收标准
- ai-prototyper：实现和实验
- code-reviewer：测试、质量、安全检查
- chief-of-staff：任务拆解和进度跟踪
