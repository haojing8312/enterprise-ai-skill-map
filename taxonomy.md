# Taxonomy｜企业场景分类体系

## 一级场景

| ID | 场景 | 典型问题 |
|---|---|---|
| office | 办公协同 | 会议、邮件、文档、日程、知识库、汇报 |
| finance-tax | 财税 | 发票、报销、预算、成本分析、税务资料准备 |
| legal-compliance | 法务合规 | 合同初审、条款对比、隐私合规、风控提示 |
| sales-growth | 销售增长 | 客户研究、线索筛选、拜访准备、跟进、CRM |
| customer-service | 客服/交付 | 工单归类、FAQ、客户成功、实施手册、复盘 |
| hr-training | 人力/培训 | 招聘 JD、简历筛选、培训课件、员工手册、考核反馈 |
| r-and-d | 研发/产品 | 需求、代码、测试、排障、文档、发布检查 |
| marketing-content | 市场/内容 | 选题、写作、图文、视频脚本、分发、数据复盘 |
| operations-admin | 行政/运营 | 采购、资产、制度、流程、跨部门协调 |
| industry-solutions | 行业方案 | 医疗、教育、制造、政务、园区、金融等 |

## 二级能力

- research / monitoring：搜集、监控、信息雷达
- drafting / generation：写作、代码、设计、内容生成
- review / compliance / QA：审核、合规、质量检查
- automation / integration：自动化执行、系统集成
- analytics / memory / reporting：复盘、知识沉淀、报告
