# Changelog

## [0.1.0] - 2026-05-31

首次公开发布。

### Added
- 企业级 AI Skill 知识库的公开仓库结构与基础规范。
- `data/skills.yaml`：结构化 Skill 总表。
- `taxonomy.md`：企业场景分类体系。
- `scenarios/`：办公、内容运营、法务合规、运营管理、研发等场景页。
- `workflows/`：内容运营数字员工工作流样板。
- `templates/`：Skill 卡片、场景页与工作流模板。

### Rules
- 仅收录公开可访问、可验证、边界清晰的内容。
- 不收录密钥、Cookie、客户信息、私有仓库、内部路径或规避平台限制的方案。
- 对安全边界不清、来源不足的条目保持审慎或暂缓收录。
